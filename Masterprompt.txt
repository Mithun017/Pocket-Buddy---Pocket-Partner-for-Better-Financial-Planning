Project - Pocket Buddy - Pocket Partner for Better Financial Planning

Build a full-stack AI-powered WealthTech web application that provides personalized financial recommendations, real-time insights, and an AI chatbot for user queries.

---

## 🎯 PROJECT GOAL

Create a scalable platform where users can:

* Sign up / log in securely
* Enter financial and personal data
* Receive AI-driven personalized investment recommendations
* View real-time financial insights (stocks, mutual funds, etc.)
* Interact with an AI chatbot for financial queries

---

## 🧩 SYSTEM ARCHITECTURE

### Frontend:

* Framework: React.js (Functional components + Hooks)
* Styling: Traditional CSS (no Tailwind or frameworks)
* State Management: Context API or Redux
* Routing: React Router

### Backend:

* Framework: FastAPI (Python)
* API Style: RESTful APIs
* Authentication: JWT-based authentication

### Database:

* MongoDB (NoSQL)
* Use PyMongo or Motor for async DB handling

### AI/ML Components:

* Recommendation Engine (Hybrid: Rule-based + ML-based)
* Chatbot (LLM API or local model)
* Real-time data scraper (financial APIs or scraping)

---

## 🔐 AUTHENTICATION MODULE

* User registration (email + password)
* Secure password hashing (bcrypt)
* JWT login system
* Role: Basic user (extendable)

---

## 👤 USER PROFILE COLLECTION

After login, collect:

* Age
* Income
* Savings
* Risk appetite (low/medium/high)
* Financial goals (short-term / long-term)
* Investment preferences

Store this in MongoDB.

---

## 🧠 PERSONALIZATION ENGINE

### Phase 1 (Rule-Based Logic):

* If risk = low → suggest bonds, low-risk mutual funds
* If risk = high → suggest stocks, aggressive portfolios
* If goal = short-term → liquid funds
* If goal = long-term → equity funds

### Phase 2 (Advanced ML):

* Use clustering (K-Means) to group users
* Collaborative filtering for recommendations
* Portfolio optimization logic

---

## 📊 RECOMMENDATION SYSTEM (DETAILED)

Implement a hybrid system:

### Inputs:

* User profile
* Market data (real-time)
* Historical performance

### Logic:

* Risk scoring formula
* Asset allocation model
* Diversification strategy

### Output:

* Personalized portfolio:

  * % allocation (stocks, bonds, mutual funds)
  * Specific recommendations
  * Expected return vs risk
